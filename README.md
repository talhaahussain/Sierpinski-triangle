# Sierpinski-triangle

A Python program to demonstrate the generation of the Sierpi≈Ñski triangle fractal using the built-in Turtle module.

![image](https://github.com/talhaahussain/Sierpinski-triangle/assets/73853725/081db3d4-592a-4cfc-9650-f55bc74ae99a)

### Usage

Run with:
```bash
python sierpinski.py
```
